# Support

Use the following channels after reviewing the README and troubleshooting section:

- GitHub Issues for reproducible bugs and focused feature requests.
- GitHub Discussions, when enabled, for usage questions, ideas, and examples.
- Private vulnerability reporting for security concerns, as described in `SECURITY.md`.

When reporting a rendering problem, include the operating system, Node.js and pnpm versions, `pnpm demo:doctor` output, the relevant demo configuration, and media metadata. Do not attach credentials, private recordings, or confidential terminal output.
